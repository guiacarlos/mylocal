MyLocal API v1

API publica para integraciones con MyLocal. Requiere API key por local.

Autenticacion

Todas las peticiones requieren el header X-Api-Key con una key valida del local.

X-Api-Key: myl_abc123...

Rate limit: 100 peticiones/minuto por key.

Base URL

POST /api/v1/{slug}/{accion}

Endpoints

POST /api/v1/{slug}/get_carta

Devuelve la carta completa del local.

{
  "success": true,
  "data": {
    "local": { "nombre": "...", "slug": "..." },
    "carta": [{ "categoria": {...}, "productos": [...] }]
  }
}

POST /api/v1/{slug}/get_table_status

Consulta el estado de una mesa.

ParametroTipoDescripcion
mesa_idstringID de la mesa
{ "success": true, "data": { "status": "free" } }

POST /api/v1/{slug}/create_order

Crea un pedido externo (delivery o integracion).

ParametroTipoDescripcion
itemsarrayLista de items [{id, name, quantity, price}]
totalnumberTotal del pedido

POST /api/v1/{slug}/get_order_status

Consulta el estado de un pedido.

ParametroTipoDescripcion
sesion_idstringID de la sesion

Errores

CodigoDescripcion
401API key no proporcionada
403API key invalida
429Rate limit superado
400Accion no permitida
